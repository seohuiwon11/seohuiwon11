## Current Project  
### Public-Toilet-Datamap

> 공공데이터를 기반으로 공중화장실 위치를 시각화하고, 장애인 화장실 여부 및 행정동별 분포 등을 직관적으로 확인할 수 있는 파이썬 기반 데이터 시각화 서비스입니다.

### GitHub Repository  
> https://github.com/yuurmj/public-toilet-datamap

### 프로젝트 목적  
>- 공중화장실 정보를 한눈에 확인할 수 있도록 지도 기반으로 시각화  
>- 장애인 화장실 여부, 행정동별 분포 등 접근성을 높이는 분석 제공  
>- 흩어진 공공데이터를 통합하여 사용자가 빠르게 필요한 정보를 찾을 수 있도록 개선  
>- 지자체 및 의사결정자에게 효율적인 인프라 관리·배치 참고 자료 제공  

## My Role in This Project  
### Data Visualization Developer  

### ✔ 주요 업무  
- **데이터 수집 및 전처리**  
  - 공공데이터 CSV 수집, 인코딩 처리, 오류 좌표 검출  
  - 좌표 유효성 검사 자동화 스크립트 개발 (`validate_coordinates.py`)  
  - 시각화용 데이터셋 가공 및 `export_for_app.py` 생성  

- **Backend Logic & Data Pipeline 구축**  
  - 정제 → 검증 → 변환 → 배포까지 이어지는 데이터 파이프라인 설계  
  - 행정동 기반 통계 자동 생성  
  - Streamlit 앱에서 사용할 구조적 데이터셋 출력  

- **Visualization & Front Logic**  
  - Streamlit을 활용한 UI 개발 (설정 페이지, 소개 페이지)  
  - UX 향상을 위한 필터, 컴포넌트 구조 개선  

- **Documentation**  
  - 프로젝트 구조 문서화  
  - 데이터 처리 흐름, 파일 역할, 모듈 간 관계 정리
